# Guest Phone - Wedding Message Recorder

A Raspberry Pi 2-based audio recorder that repurposes an antique phone for wedding guest messages.

## Overview

This project transforms a vintage telephone into an interactive wedding guestbook. Guests pick up the phone, hear a pre-recorded message and tone, then leave congratulatory messages for the newlyweds. The system uses the phone's original microphone and speaker for an authentic experience.

## Hardware

- Raspberry Pi 2
- Antique telephone (with original mic and speaker)
- Audio interface (to be determined)

## Features

- Pre-recorded greeting message
- Audio tone to signal recording start
- Records guest messages
- Uses original phone hardware for authentic feel
